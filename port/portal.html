行程庫管理

電腦版部署教學

1. 點選「建立副本」存入雲端硬碟。

2. 開啟試算表，於 SYSTEM 填寫金鑰與設定。

3. 點選 擴充功能 > Apps Script。

4. 點選右上角 部署 > 新部署，對象選「所有人」。

5. 完成授權並複製 Web App 網址貼回本頁。

手機部署教學

點我觀看手機操作示範影片
1

複製副本與切換模式

點擊「取得副本」後，請點瀏覽器選單並勾選 「要求電腦版網站」。這非常重要，否則無法進入 Script 編輯器。

2

執行初始化腳本

進入 擴充功能 > Apps Script，在左側選單點擊 setup.gs。點擊上方的 「執行」。

授權流程： 審查權限 > 選帳號 > 進階 > 前往 (不安全) > 全選權限 > 允許。
3

正式部署網址

點選右上角 部署 > 新部署。 設定「誰有權限存取」為 「所有人」。最後再次完成授權並複製生成的網址。

4

填寫資料夾 ID

回到試算表的 System 分頁，確認金鑰與腳本自動產生的資料夾 ID 是否正確，再回本頁提交。
